DESCRIPCIÓN DEL PROYECTO

Este proyecto es un Tres en Raya (Tic Tac Toe) desarrollado con React como primer contacto práctico con este framework.

El objetivo principal no es únicamente crear el juego, sino aprender React desde la práctica, comprendiendo conceptos como el uso de componentes, el manejo del estado, el renderizado condicional y la organización de un proyecto React.

Este proyecto forma parte de una serie de proyectos pequeños cuyo propósito es aprender lo máximo posible sobre React mediante la práctica constante. Es el primero de muchos.

---

OBJETIVO DEL PROYECTO

- Tener un primer acercamiento práctico a React
- Comprender la estructura básica de un proyecto React
- Practicar el uso de componentes y estado
- Aprender a través de proyectos pequeños y progresivos

DESCRIPCIÓN DE LAS CARPETAS

src/components  
Contiene los componentes reutilizables del juego.  
Square representa cada casilla del tablero.  
Board gestiona el tablero y la lógica del juego.

App.jsx  
Componente principal donde se controla el estado general del juego y se renderiza la aplicación.

main.jsx  
Punto de entrada de la aplicación React.
